Funções

para que um evento seja executado em JS eu preciso coloca-lo em uma função = function

Para que o HTML não fique poluido eu posso usar o ouvidor addEventListener

Se eu colocar o addEventListener dentro da função não funciona (não estava usando vAR fora da função, mas o LET em cada função).
No caso desse exercício que eu estou treinando o ouvidor addEventListener só funciona quando está de forma global tanto como LET ou VAR
